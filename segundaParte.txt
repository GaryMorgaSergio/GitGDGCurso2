Sesion 2:
    - Adicion de Archivos
        - git add (nombre del archivo con su extensi√≥n);
        - git add * (Adiciona todos los archivos) 
    - Status del repositorio (commit, agregaciones, las ramas)
        - git status
    - Commit, nos permite agrupar un conjunto de cambios 
        - git commit -m ("mensajes para el commit")
    - Enlazar nuestro repositorio remoto al repositorio local 
        - git remote add origin  (url del repositorio)
        - git rmote -v (permite ver el repositorio remoto al que estamos enlazados)
    - Subir cambios al repositorio remoto
        - git push 
    - Log ( observar los commit realizados)
        - git log
    - Restauracion de un archivo 
        - git restore --staged (nombre del archivo a restaurar)
    - Navegar a un commit anterior (Cuidado con este comando)
        - git reset --hard (id del commit al que quiere navegar)
